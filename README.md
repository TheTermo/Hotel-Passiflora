# Projekt końcowy 

Strona hotelu z systemem rezerwacji 

https://hotel-passiflora-booking-site-react.netlify.app/

## Funkcje

- Przeglądanie pokoi

- Informacje o udogodnieniach i różne info o hotelu

## Technologie

Projekt został zbudowany przy użyciu następujących technologii:

- React
- Material UI
- SASS
